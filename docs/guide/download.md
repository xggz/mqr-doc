## 其它平台
其它平台目前只提供可运行的jar包，需要java环境，并使用java命令启动程序（当然windows也可以使用此方式运行）。

下载地址：[https://pan.baidu.com/s/1s5iycwWXzp68i3SdPbiwbw](https://pan.baidu.com/s/1s5iycwWXzp68i3SdPbiwbw)

提取码：45in

简单启动命令示例：

``` bash
java -jar mqr.jar
```

Linux后台静默运行，并输出日志
``` bash
nohup java -jar mqr.jar > mqr.log 2>&1 &
```
